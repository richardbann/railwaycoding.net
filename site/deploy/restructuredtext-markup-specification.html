Quick Syntax Overview

Paragraphs

Paragraphs contain text and may contain inline markup: emphasis, strong emphasis, interpreted text, inline literals, standalone hyperlinks (http://www.python.org), external hyperlinks (Python), internal cross-references (example), footnote references ([1]), citation references ([CIT2002]), substitution references (example), and inline internal targets, which can be referred later: inline internal targets.

Options list

-a command-line option "a"

--long options can be long also
--input=file long options can also have arguments
/V DOS/VMS-style options too

Literal Blocks

Literal blocks are either indented or line-prefix-quoted blocks, and indicated with a double-colon ("::") at the end of the preceding paragraph (right here -->):

if literal_block:
    text = 'is left as-is'
    spaces_and_linebreaks = 'are preserved'
    markup_processing = None

Block Quotes

Block quotes consist of indented body elements:

This theory, that is mine, is mine.

—Anne Elk (Miss)
